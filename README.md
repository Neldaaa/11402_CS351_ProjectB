# Project B - CSV Mini Databse & Query Engine

## Overview

This project implements a lightweight CSV processing system with query capabilities. It enables users to load CSV files, perform basic data operations, and execute queries similar to SQL against the dataset. The system handles parsing, storage, filtering, and aggregation of tabular data without requiring a full database engine.

## Features

- **CSV File Loading**: Parse and import CSV data with support for various delimiters
- **Data Storage**: Maintain in-memory representation of tabular data
- **Query Engine**: Execute filtering, sorting, and aggregation operations
- **Data Manipulation**: Support for basic operations like selection, projection, and joins

## Objectives

- Build a functional query processor for flat-file data
- Understand data structure design and algorithm optimization
- Implement basic database operations from first principles
